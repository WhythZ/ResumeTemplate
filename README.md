## About
This repo is a minimalistic resume template by LaTeX for job hunting, feel free to swap out the content for your own details or extend the template as needed

![Example.png](https://github.com/WhythZ/ResumeTemplate/blob/master/Example.png)

## Environment
To edit the LaTeX project in real-time by VSCode, you should install following dependencies
- The [TexLive](https://tug.org/texlive/acquire-iso.html) without `TeXworks` frontend
- The `LaTeX Workshop` plugin in VSCode
